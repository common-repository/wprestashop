== Description ==
You can integrate wordpress and prestashop perfectly.
we can call all block in prestashop:
 Block Category
 Top sellers 
 New products
 Featured products
 Block Cart 
 Block UserInfo

 
== Installation ==
1. Upload `wprestashop` to the `/wp-content/plugins/` directory and unzip
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click refresh if vimeo quicktag does not show after activation.
